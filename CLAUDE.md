# MCP Servers

Custom MCP servers for personal use (Gmail, Google Calendar, Google Drive, Apple Notes, etc.).

## Trust Policy

These integrations handle sensitive personal data. Only use code from:

1. **Anthropic** - Official MCP examples and libraries
2. **The product owner** - e.g., Google for Gmail/GCal/GDrive, Apple for Notes/Messages

Do NOT use random GitHub projects or third-party implementations for these integrations, regardless of popularity or convenience.

## Current Servers

- `gmail-mcp-server` - Gmail integration
- `gcal-mcp-server` - Google Calendar integration
- `gdrive-mcp-server` - Google Drive integration
- `apple-notes-mcp-server` - Apple Notes integration
